# 🎨 **초등학생을 위한 스케치북 사용법 가이드** 🎨

---

## 📱 **1. 스케치북이 뭐예요?**
스케치북은 **컴퓨터나 태블릿에서 그림을 그릴 수 있는 마법 같은 프로그램**이에요!

👉 **장점:**
- 지우개를 잘못 눌러도 걱정 없어요! (되돌리기 버튼이 있어요)
- 색깔을 마음대로 바꿀 수 있어요!
- 레이어(투명한 종이)를 여러 장 겹쳐서 쓸 수 있어요!

🎨 **시작 방법:**
1. **스케치북 앱**을 열고 **"새 그림"**을 선택해요.
2. **캔버스**가 나타나면 그림을 그릴 준비가 완료된 거예요!

---

## 🛠️ **2. 기본 도구 설명 (그림 도구들)**

| **아이콘** | **이름** | **어떻게 쓰는 거야?** | **어디서 찾나요?** |
|------------|----------|------------------------|---------------------|
| ✏️ | **연필** | 스케치할 때 쓰는 연필 같아요! | 왼쪽 도구 상자에서 선택 |
| 🖌️ | **브러시** | 물감 붓 같아요! 굵고 부드러운 선을 그릴 수 있어요 | 브러시 아이콘 클릭 |
| 🖊️ | **펜** | 뚜렷한 선을 그릴 때 좋아요 | 펜 아이콘 선택 |
| 🧽 | **지우개** | 잘못 그린 부분을 지워요! | 지우개 아이콘 클릭 |
| 🖱️ | **선택 도구** | 그림을 움직이거나 복사할 때 써요 | 선택 아이콘 선택 |

💡 **Tip:** 도구는 **마우스나 손가락으로 클릭**해서 선택해요!

---

## 🖌️ **3. 브러시 사용법 (붓질하는 방법)**

1️⃣ **브러시 선택하기**
   - 상단 메뉴에서 🖌️ **브러시 아이콘**을 누르세요.
   - 여러 가지 브러시가 있어요! (예: 연필, 펜, 스프레이, 물감)

2️⃣ **크기 조절하기**
   - **브러시 크기 슬라이더**를 움직여서 굵기를 조절해요.
   - 🔍 **작게** → 세밀한 그림
   - 🔍 **크게** → 넓은 면적 칠하기

3️⃣ **색상 선택하기**
   - 왼쪽 아래 **색상 원**을 누르면 색상 팔레트가 나와요.
   - 좋아하는 색을 골라서 브러시로 칠해요!

4️⃣ **그리기 시작!**
   - 캔버스 위에서 **드래그(누르고 움직이기)** 하면 그림이 그려져요!

🎨 **연습하기:**
- 동그라미, 네모, 별을 그려보세요!
- 색을 바꿔가며 그림을 완성해 보세요!

---

## 📚 **4. 레이어 (투명한 종이) 관리하기**

레이어는 **투명한 종이 여러 장을 겹쳐서 쓰는 것**이라고 생각하면 돼요!

📌 **레이어 사용법:**

1️⃣ **새 레이어 만들기**
   - 오른쪽 패널에서 **"+" 버튼**을 누르세요.
   - 새로운 투명한 종이가 생겨요!

2️⃣ **레이어 이름 바꾸기**
   - 레이어 이름을 **더블클릭**해서 원하는 이름으로 바꿔요.
   - 예: "하늘", "땅", "별"

3️⃣ **레이어 순서 바꾸기**
   - 레이어를 **드래그(누르고 움직이기)**해서 순서를 바꿀 수 있어요.
   - 🌈 **아래에 있는 레이어**가 **위에 있는 레이어**를 덮어요!

4️⃣ **레이어 숨기기/보이기**
   - 레이어 왼쪽의 **눈 아이콘**을 클릭하면 숨겼다 보였다 해요.

💡 **예시:**
- **1번째 레이어:** 하늘 (파란색)
- **2번째 레이어:** 구름 (흰색)
- **3번째 레이어:** 나무 (초록색)

---

## 🎨 **5. 색상 선택하는 방법**

색상을 고르는 방법은 **매우 쉽고 재미있어요!**

1️⃣ **색상 팔레트 열기**
   - 왼쪽 아래 **색상 원**을 누르세요.

2️⃣ **색상 고르기**
   - **원형 팔레트**에서 원하는 색을 선택해요.
   - **RGB 슬라이더**를 움직여서 더 정확한 색을 만들 수 있어요.

3️⃣ **최근 사용 색상**
   - 이전에 사용한 색이 **아래에 저장**되어 있어요!

4️⃣ **스포이드 도구 (색상 흡수기)**
   - 🖱️ **스포이드 아이콘**을 선택한 다음, 캔버스에서 색을 클릭하면 그 색을 바로 사용할 수 있어요!

🎨 **색상 팁:**
- **밝은 색** → 그림이 환해져요!
- **어두운 색** → 그림에 깊이감이 생겨요!
- **반짝이는 색** → 별이나 보석을 그릴 때 사용해요!

## 💾 **6. 그림 저장하기 (내 그림을 보관하자!)**

그림을 다 그렸으면 **반드시 저장**해야 해요!

1️⃣ **저장하기 (PSD 형식)**
   - **Ctrl + S** (키보드에서 S를 누르면서 Ctrl 키도 같이 누르기)
   - 또는 **파일 → 저장** 클릭
   - **PSD 형식**으로 저장하면 나중에 다시 편집할 수 있어요!

2️⃣ **다른 형식으로 내보내기**
   - **PNG** → 투명한 배경으로 저장 (배경이 없는 그림)
   - **JPG** → 일반적인 사진 형식
   - **PDF** → 문서로 저장해서 친구에게 보내기

📌 **저장 팁:**
- **이름을 지어주세요!** (예: "나의 멋진 그림.png")
- **폴더를 만들어서 정리**하면 나중에 찾기 쉬워요!

## ❓ **자주 묻는 질문 (FAQ)**

**Q1. 실수로 지운 그림을 되돌리고 싶어요!**
➡️ **Ctrl + Z**를 누르면 **한 단계 되돌리기**가 돼요!

**Q2. 브러시가 안 움직여요!**
➡️ **브러시 크기**를 확인하거나 **다시 선택**해 보세요!

**Q3. 그림이 너무 작아요!**
➡️ **확대/축소 버튼**을 눌러서 크기를 조절하세요!

**Q4. 레이어가 너무 많아요!**
➡️ **필요 없는 레이어는 삭제**하거나 **이름을 바꿔서 정리**하세요!
